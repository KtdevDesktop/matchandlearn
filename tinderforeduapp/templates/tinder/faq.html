  • พื้นฐาน

  • ฉันจะสามารถสมัครสมาชิก Match and Learn ได้อย่างไร

    ในการสมัคสมาชิก ให้คุณเลือก New to Match and Learn? Sign up now! จากนั้นทำการกรอกข้อมูลต่างๆ แล้วกด Sign up จากนั้นให้คุณไปที่ Email ที่คุณได้สมัครไว้ เพื่อทำการยืนยันการสมัครสมาชิก ทั้งนี้ คุณสามารถใช้ Continue with Facebook ในการสมัครสมาชิกได้ทันที

  • ฉันจะสามารถเข้าสู่ระบบได้อย่างไร

    ในการเข้าสู่ระบบ ให้คุณกรอก Username และ Password ที่ใช้ในการสมัครสมาชิก จากนั้นกด Log in ในกรณีที่คุณสมัครสมาชิกด้วย Facebook คุณสามารถกด Continue with Facebook ได้เลย โปรดจำไว้ว่า ถ้าหากว่าคุณสมัครสมาชิกด้วย Facebook คุณจะต้อง Login ด้วยปุ่ม Continue with Facebook เท่านั้น

  • ฉันลืมรหัสผ่าน จะทำอย่างไรดี

    หากคุณลืมรหัสผ่าน ให้คุณกด Forgot password? ที่หน้า Login จากนั้นใส่ Email ที่ได้ทำการสมัครสมาชิกไว้ ถ้าหากว่าระบบตรวจพบ User ที่สมัครผ่าน Email ที่กรอกเข้ามา ระบบ จะส่งลิ้งค์กู้คืนรหัสผ่านไปให้ทาง Email ให้คุณคลิกที่ลิ้งค์นั้น เพื่อทำการเปลี่ยนรหัสผ่าน

  • ฉันไม่ได้ทำการสมัครสมาชิกด้วย Facebook ฉันจะสามารถใช้ปุ่ม Continue with Facebook ได้หรือไม่

    สามารถทำได้ ในการกด Continue with Facebook ถ้าระบบตรวจพบว่า Email ของบัญชี Facebook ของคุณ ตรงกับ Email ของ User ที่เคยสมัครเอาไว้ ระบบ จะทำการ Log in User นั้นให้โดยอัตโนมัต

  • การใช้งานระบบ Match

  • ฉันจะค้นหา Tutor ได้อย่างไร

    คุณสามารถค้นหา Tutor ที่จะสามารถสอนคุณได้โดยการระบุวิชาที่คุณต้องการจะหา Tutor ลงในช่อง Enter a Subject ที่หน้า Home และระบบจะทำการแสดง User ทั้งหมดที่ระบุวิชาที่คุณค้นหาเป็นวิชาที่ถนัด

  • ฉันค้นหาเจอ Tutor ที่ฉันต้องการจะ Match แล้ว ฉันต้องทำอย่างไรต่อ

    เมื่อคุณค้นหาจนเจอ Tutor ที่คุณต้องการจะ Match ด้วยแล้ว ให้คุณไปที่หน้า Profile ของ Tutor คนนั้น และกดปุ่ม Match ระบบ จะทำการแจ้งเตือนไปยัง User คนที่คุณกด Match ว่าคุณต้องการที่จะ Match กับเขา/เธอ

  • ฉันจะดู Profile ของตัวเองได้อย่างไร

    ในการดู Profile ของตัวเอง ให้คุณคลิกที่ Profile : (ชื่อจริงของคุณ) จากนั้นคุณจะสามารถตรวจสอบ Profile ของคุณ ในหน้านี้ คุณสามารถที่จะเพิ่ม หรือ ลบ วิชาที่ถนัดได้ และยังสามารถแก้ไข Profile ในส่วนอื่นๆ ได้อีกด้วย

  • มีคนขอ Match ฉันมา ฉันจะต้องทำอย่างไรต่อ

    เมื่อมีคน Request Match มา คุณจะได้รับการแจ้งเตือนที่แถบด้านบน คุณสามารถคลิกเข้าไปเพื่อตรวจสอบว่า User คนใดเป้นคนส่ง Request มา เมื่อคุณเข้าไปในหน้า Profile ของคุณที่ส่ง Request มาแล้ว คุณสามารถเลือกได้ว่าคุณจะยอมรับคำขอ (Accept) หรือปฏิเสธคำขอ (Decline) ในกรณีที่คุณยอมรับคำขอ User ที่ส่ง Request มา จะถือว่าเป็น Student และคุณจะเป็น Tutor ของ User ที่ส่ง Request มาทันที

  • การใช้งานระบบ Chat

  • ฉัน Match กับ Tutor/Student ของฉันแล้ว ฉันจะคุยกับอีก User นึงได้อย่างไร

    เมื่อคุณต้องการที่จะใช้ระบบ Chat ให้คุณคลิกที่หัวข้อ Chat ที่แถบด้านบน จากนั้นในหน้า Chat ระบบจะแสดงรายชื่อของ User ที่คุณสมารถที่จะ Chat ด้วยได้ โดยรวมทั้ง Tutor และ Student โดยเมื่อคุณคลิกที่ User ที่ต้องการ ระบบจะพาไปยังหน้า Chat ทันที
